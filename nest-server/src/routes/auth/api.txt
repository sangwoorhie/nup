1. 회원가입 (개인회원) 
HTTP Method: POST
URL: localhost:3000/auth/signup1
Description: 개인회원 가입을 위한 API
Request Body:
json
-
{
  "email": "string",
  "password": "string",
  "confirmPassword": "string",
  "username": "string",
  "phone": number,
  "emergency_phone": number (optional),
  "profile_image": "string" (optional)
}
Response:
Status Code: 201
Body:
json
-
{
  "id": "string",
  "accessToken": "string",
  "refreshToken": "string"
}

3. 로그인 (email, password)

Request Body:
json
-
{
  "email": "string",
  "password": "string"
}
Response:
Status Code: 200
Body:
json
-
{
  "accessToken": "string",
  "refreshToken": "string"
}



4. 로그인 (apiKey)

json
-
{
  "apiKey": "string",
}
Response:
Status Code: 200

json
-
{

}



5. 리프레시토큰 발급
HTTP Method: POST
URL: localhost:3000/auth/refresh
Description: 리프레시토큰 발급을 위한 API
Request Headers:
Authorization: Bearer {refreshToken}
Request Body:
json
-
{
  "accessToken": "string",
  "refreshToken": "string"
}
Response:
Status Code: 200
Body:
json
-
{
  "accessToken": "string",
  "refreshToken": "string"
}



 6. 로그아웃
HTTP Method DELETE : 
URL : localhost:3000/auth/signout

Authorization: Bearer [JWT 토큰]
